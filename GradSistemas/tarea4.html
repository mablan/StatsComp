2. Algunas personas afirman que los estudiantes ms jvenes se gradan con mejores promedios que los estudiantes mayores. Para este grupo de datos en particular, realice un grfico que permita visualizar si es o no cierta esta afirmacin. Extra puntos si a su grfico le sobrepone una curva describiendo la tendencia (Indicacin: En el vdeo visto en clase se muestra como hacer esto utilizando loess).

xyplot(prom_general~ao_nacimiento,data=grad,pch=20,
       panel = function(x, y, ...) {
         panel.xyplot(x, y, ...)
         panel.loess(x, y) }
)

Mirando este grfico parece que, en efecto, los estudiantes ms jvenes se graduan con mejores promedios que los estudiantes mayores.
